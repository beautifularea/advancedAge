## tcpdump  
1. 过滤指定端口数据   
sudo tcpdump -i lo port 5060 -X -vv  

-i 指定监听的网络接口</br>
-X 显示数据包内容（HEX/ASCII)</br>  
-vv 输出详细的报文信息</br>
port 指定端口号</br>
